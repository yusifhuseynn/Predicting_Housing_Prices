 Predicting Housing Prices 🏠

## 📌 Project Overview
This project is part of IndoLike's Machine Learning tasks.  
**Objective:** Build a predictive model to estimate housing prices based on key features such as location, size, and age.  
We use the **Boston Housing Dataset** to train, evaluate, and compare different regression models.

---

## 📂 Dataset
- **Source:** [Boston Housing Dataset](https://raw.githubusercontent.com/selva86/datasets/master/BostonHousing.csv)  
- **Target Variable:** `MEDV` – Median value of owner-occupied homes in $1000's  
- **Features:** Includes variables like crime rate, average rooms, property tax rate, and more.

---

## 🛠️ Technologies Used
- **Python 3**
- **Pandas, NumPy** – Data handling & preprocessing
- **Matplotlib, Seaborn** – Data visualization
- **Scikit-learn** – Machine learning models & evaluation

---

## 🔍 Project Workflow
1. **Data Loading** – Import dataset into Pandas DataFrame
2. **Exploratory Data Analysis (EDA)** –  
   - Summary statistics  
   - Target variable distribution  
   - Correlation heatmap
3. **Data Preparation** –  
   - Splitting into training and testing sets  
   - Normalization
4. **Model Training** –  
   - Linear Regression  
   - Random Forest Regressor  
   - Gradient Boosting Regressor
5. **Model Evaluation** –  
   - RMSE (Root Mean Squared Error)  
   - MAE (Mean Absolute Error)  
   - R² Score
6. **Comparison & Conclusion** – Identify the best performing model
